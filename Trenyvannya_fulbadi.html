Рoзглянемо один з типажів поділу тренувань за групами м'яз - фулбаді

Тренування фулбаді – це система вправ, яка дозволяє тренувати всі основні м'язи тіла за один підхід. Зазвичай такі тренування проводять 2-4 рази на тиждень, що дає можливість м'язам відновлюватись між сесіями. У програмі зазвичай включаються базові вправи, які задіюють велике число м'язових груп та суглобів.

  1. Розминка
  2. Присяд з власною вагою або штангою — 3 підходи по 10-15 повторень.
  3. Жим лежачи з гантелями або штангою — 3 підходи по 8-12 повторень
  4. Станова тяга (традиційна або румунська) — 3 підходи по 8-10 повторень
  5. Тяга до поясу з гантелями або штангою — 3 підходи по 8-12 повторень.
  6. Планка — 3 підходи по 30-60 секунд.
  7. Заминка
